# Contributor Covenant Code of Conduct

FastShield is committed to providing a welcoming and harassment-free experience for everyone. Examples and expectations are intentionally brief here — follow the full Contributor Covenant at https://www.contributor-covenant.org.

Be respectful, courteous, and professional. Do not harass or discriminate. If you witness or experience unacceptable behavior, please report it to the repository maintainers via a GitHub issue or contact the project owner directly.

This project follows the Contributor Covenant v2.1.
